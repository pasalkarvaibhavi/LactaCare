### LactaCare - Human Milk Bank Management System
---
**LactaCare** is a web-based human milk management system designed to connect human milk **donors**, **hospitals**, and **administrator** on a single platform to support infants in need of breastmilk. It allows donors to register, complete health screenings, book appointments, and track donation history, while hospitals can
request milk for newborns and monitor request status. Admins manage appointments, milk requests, and inventory - making the entire milk donation and distribution process efficient, transparent,
and life-saving for vulnerable infants.


#### Technologies Used

- **Backend:** Java, JSP, Servlets, JDBC
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** PostgreSQL
- **Server:** Apache Tomcat
- **Security:** jbcrypt (for secure password hashing)
- **IDE:** Eclipse
